# CS50W Projects

A collection of web applications built for [Harvard's CS50W: Web Programming with Python and JavaScript](https://cs50.harvard.edu/web/).

Each project lives on its own branch:

| Branch | Description |
|--------|-------------|
| `capstone` | HireFlow - Internship management SaaS - Final capstone project |
| `commerce` | stockX-like auction site |
| `mail` | Single-page email client |
| `network` | My take on a social network - Spire |
| `search` | Google Search front-end - first project |
| `wiki` | Wikipedia-like encyclopedia |

## Viewing a Project

Switch to the relevant branch to explore a project's code:
```bash
git checkout <branch-name>
```
